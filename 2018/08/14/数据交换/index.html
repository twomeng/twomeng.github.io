Description

给你2个长度相同的字符串数组，请交换2个数组的内容。
本题要求必须使用函数传参来实现!

Input

输入数据有多组，每组第1行为n( 2<=n<=10),接下来输入2行，每行都为n个字符串（串的长度小于20）;

Output

请输出交换后的2组串。

Sample Input

3 
crise tear tire 
base bind dipaloma

Sample Output

base bind dipaloma
crise tear tire
